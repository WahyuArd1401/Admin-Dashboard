# React + Vite

## Tech Stack
- React Router
- Lucide React
- Framer Motion

## How to use
1. Clone Project
```yaml
git clone https://github.com/WahyuArd1401/Admin-Dashboard.git
```
2. Install package
```yaml
npm install
```
3. Run program
```yaml
npm run dev
```
